## LINQ Practice Project

This project is a practice exercise focused on **LINQ (Language Integrated Query)** in C#.

It contains a collection of queries implemented on in-memory data, based on the assignment file:

**"LINQ to Objects Assignment"**

### 📌 Purpose

The goal of this project is to:

* Practice writing LINQ queries
* Understand different LINQ operators
* Work with collections using `LINQ to Objects.`

### 📂 Contents

* Multiple examples of LINQ queries
* Filtering, sorting, grouping, and projection operations
* Hands-on solutions to assignment problems

### 🛠 Technologies Used

* C#
* .NET
* LINQ
